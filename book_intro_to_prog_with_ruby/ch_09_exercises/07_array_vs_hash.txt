7. What's the major difference between an Array and a Hash?

array is a list of objects stored in indexes.
referencable by index value.
store objects in an array when sorting matters.
objects can be any type: string, integer, float, symbols, hash etc.

Hash is objects stored in key value pairs. and referencable by keys.
store objects in an hash when sorting doesn't matter.
keys can only be of type: string, integer, symbols, hash etc.
